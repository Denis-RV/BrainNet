## Создание таблицы
Таблица создается с помощью команды `CREATE TABLE`. Упрощенный синтаксис:
```sql
CREATE TABLE table_name
(
	column_name data_type [column_constraint],
	column_name data_type [column_constraint],
	...
	column_name data_type [column_constraing],
	[table_constraint],
	[первичный ключ],
	[внешний ключ]
);
```
